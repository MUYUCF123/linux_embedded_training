# Day4 学习总结

## 1. 我做了什么
### 简易状态机

为了准备自己的原创项目，今天练习了简易状态机的实现，主要是理解了：

1. 抛弃原来成堆if和flag的思维，利用枚举类型和指针，分别定义状态和事件，switch case来处理状态转移，分状态处理事件，十分清晰
2. 对于事件的区分，之前一直都是：判断按键按下，直接处理对应逻辑，现在将其归为外部输入事件，就是把事件类型分开，比如按键事件就叫EVENT_KEY，而处理事件则是EVENT_PLAY，这样就更清晰了

### 阅读了C与指针第一章

第一章主要还是基础知识，譬如：getchar函数，gets函数，以及通过while释放掉输入缓冲区的内容，避免后续输入被干扰，例如：

```c
while ((ch = getchar()) != '\n' && ch != EOF);
```
